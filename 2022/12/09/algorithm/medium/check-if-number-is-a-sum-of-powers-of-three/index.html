LC1780·判断一个数字是否可以表示成三的幂的和

题目

1780. 判断一个数字是否可以表示成三的幂的和

给你一个整数 n ，如果你可以将 n 表示成若干个不同的三的幂之和，请你返回 true ，否则请返回 false 。

对于一个整数 y ，如果存在整数 x 满足 y == 3x ，我们称这个整数 y 是三的幂。

示例 1：

输入：n = 12
输出：true
解释：12 = 31 + 32

示例 2：

输入：n = 91
输出：true
解释：91 = 30 + 32 + 34

示例 3：

输入：n = 21
输出：false

提示：

  • 1 <= n <= 10^7

题解

深度优先搜索

判断若干个数之和是否等于 n 且这些数不得重复出现。

  • 计算出小于 n 的3的任意幂，存入集合中。
  • 判断集合中元素和是否可以等于 n ，切不得重复。
  • 用 n 减去集合的元素，直到差为0。

复杂度

  • 时间 $$O ( n ^ { 1 / \operatorname { log } _ { 2 } 2 ^ { 3 } } )$$ 其中 二叉树共有个 $$2^m - 1$$ 个节点， m 为树的深度，$$m = \operatorname { log } _ { 3 } n$$。

  • 空间 $$O ( n ^ { 1 / \operatorname { log } _ { 2 } 2 ^ { 3 } } )$$ 所有节点均要入队

三进制

该题最优解解法

将十进制数转化为3进制数即可

请尝试用二进制去理解这个世界…

复杂度

  • 时间O(logn)
  • 空间O(1)
